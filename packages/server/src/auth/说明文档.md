/api/auth/telegram 这三份文件的登录接口调用顺序应为：

/api/auth/telegram

1. verifyTelegramInitData(initData)
2. 调用 Supabase RPC：auth_upsert_telegram_user
3. issueSession({ userId, telegramUserId, initDataAuthDate, initDataHash })
4. 写入 HttpOnly Cookie

数据库 core.app_sessions 至少需要这些字段：

id uuid primary key
user_id uuid not null
telegram_user_id text null
token_hash text not null
issued_at timestamptz not null
expires_at timestamptz not null
last_seen_at timestamptz null
revoked_at timestamptz null
ip_hash text null
user_agent_hash text null
init_auth_date timestamptz null
init_hash text null
metadata jsonb not null default '{}'
created_at timestamptz not null default now()
updated_at timestamptz not null default now()
